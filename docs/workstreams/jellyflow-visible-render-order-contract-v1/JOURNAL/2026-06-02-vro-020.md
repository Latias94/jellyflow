# VRO-020 Runtime Visible Render Order Contract

Status: DONE
Date: 2026-06-02

## Summary

Added a headless runtime/store contract for ordered visible node ids:

- `runtime::rendering::resolve_visible_node_render_order`
- `NodeGraphStore::visible_node_render_order(viewport_size)`

The implementation intentionally composes existing helpers instead of duplicating logic:

- `resolve_visible_node_ids` owns viewport culling and hidden-node filtering;
- `resolve_node_render_order` owns draw order and selected-node elevation;
- the new helper filters render order through the visible-id set.
