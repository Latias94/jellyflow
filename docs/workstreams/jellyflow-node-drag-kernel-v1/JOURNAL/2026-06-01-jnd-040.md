# JND-040 Session Journal - Snap And Extent Constraints

Date: 2026-06-01
Status: DONE

## Scope

Implemented snap-to-grid and movement extent handling for the renderer-neutral node drag kernel.

## Changes

- Added shared snap-offset planning for multi-selection drag.
- Added internal drag candidates carrying node id, current position, size, resolved extent, and
  whether the node owns an extent override.
- Added global `node_extent` group clamping for multi-selection drag.
- Added per-node rect extent clamping.
- Added node-origin-aware bounds math for extent clamping.
- Added basic `NodeExtent::Parent` resolution to the parent group rect when `expand_parent` is
  false.
