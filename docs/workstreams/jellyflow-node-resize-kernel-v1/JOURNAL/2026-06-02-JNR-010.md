# JNR-010 - Workstream Opening

Date: 2026-06-02

Opened `jellyflow-node-resize-kernel-v1` as the next XyFlow-feel headless runtime lane.

Key evidence:

- XyFlow's `xyresizer` contains framework-independent resize math for dimensions, min/max bounds,
  node origin, parent extent, child extent, and keep-aspect-ratio behavior.
- Jellyflow already has `Node.size`, `GraphOp::SetNodeSize`, XyFlow size change conversion, and
  size projection, but no runtime resize planner.

Next task: JNR-020, focused tests and minimal single-node resize planning.
