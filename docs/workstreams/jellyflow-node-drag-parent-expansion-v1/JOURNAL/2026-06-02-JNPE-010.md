# JNPE-010 - Workstream Opening

Date: 2026-06-02

Opened `jellyflow-node-drag-parent-expansion-v1` as the next node-drag architecture lane.

Key evidence:

- XyFlow `updateNodePositions` collects `ParentExpandChild` entries when `expandParent` and
  `parentId` are set.
- XyFlow `handleExpandParent` expands parent rects and compensates non-dragged siblings for left/top
  parent movement.
- Jellyflow currently resolves `NodeExtent::Parent` with `expand_parent = true` to no extent and
  emits only `SetNodePos` operations.

Next task: JNPE-020, focused tests and minimal single-parent expansion planning.
