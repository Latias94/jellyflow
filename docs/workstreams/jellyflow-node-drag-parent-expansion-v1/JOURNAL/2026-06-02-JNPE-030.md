# JNPE-030 - Multi-Parent And Left/Top Expansion Contract

Date: 2026-06-02

Added focused contract tests for the behavior that remained after JNPE-020.

Changes:

- Added a multi-selection test proving node movement ops stay sorted by `NodeId` and parent
  expansion ops stay sorted by `GroupId`.
- Added a left/top expansion test proving group rect origin/size can expand while non-dragged
  sibling nodes keep their canvas-space positions.
- Documented the coordinate decision: Jellyflow node positions are canvas-space absolute, so XyFlow
  sibling compensation is not represented as extra runtime `SetNodePos` ops.

Validation:

- `cargo fmt --check` passed.
- `cargo nextest run -p jellyflow-runtime drag_parent_expansion` passed, 4 tests run.
- `cargo nextest run -p jellyflow-runtime drag` passed, 42 tests run.
