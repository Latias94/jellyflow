# JRP-050 - Store Facade Deepening

Date: 2026-05-30

## Changes

- Preserved `NodeGraphStore` as the public facade in `runtime/store.rs`.
- Moved dispatch, undo/redo, validation, middleware, and commit publication internals to `runtime/store/dispatch.rs`.
- Moved event emission helpers to `runtime/store/events.rs`.
- Moved selector/subscription bookkeeping to `runtime/store/subscriptions.rs`.
- Moved graph/view/config accessors and mutators to `runtime/store/view.rs`.
