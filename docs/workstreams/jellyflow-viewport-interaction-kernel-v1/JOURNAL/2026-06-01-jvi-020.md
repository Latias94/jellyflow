# JVI-020 Journal - Viewport Kernel

Date: 2026-06-01

## Scope

- Added public `runtime::viewport` helpers for renderer-neutral viewport transforms.
- Kept raw platform input, renderer smoke tests, animation, and callback wiring out of this task.

## Implementation Notes

- `ViewportTransform` stores canvas-space pan and positive finite zoom.
- Projection follows the existing fit-view convention: `(canvas + pan) * zoom`.
- `ViewportPanRequest` converts logical screen-pixel deltas into canvas-space pan changes by
  dividing by zoom.
- `ViewportZoomRequest` clamps target zoom and preserves the canvas coordinate under the requested
  screen anchor.
- Helpers reject non-finite pan/delta/anchor values and non-positive or non-finite zoom values.
