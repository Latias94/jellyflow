# JGS-030 Derived Layout And Spatial Queries

Status: DONE
Date: 2026-06-01

## Summary

Added parent/child lookup helpers on `NodeGraphLookups`:

- `parent_for_node`
- `child_nodes_for_parent`
- `child_nodes_by_parent`
- `root_nodes`

These are derived from `node_lookup` on demand and sort returned node ids. This avoids adding a
new public struct field to `NodeGraphLookups`.

Strengthened node spatial query tests for deterministic linear fallback behavior, hidden nodes,
and missing node sizes with fallback dimensions.

`NodeGraphSpatialIndexTuning` remains serialized config for a future indexed backend. Current node
spatial queries intentionally use linear scans plus sorted output.
