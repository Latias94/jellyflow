# JGS-020 Shared Geometry Proof

Status: DONE
Date: 2026-06-01

## Summary

Introduced `crates/jellyflow-runtime/src/runtime/geometry.rs` as the shared internal home for
canvas bounds, node-origin projection, rect union/intersection/contains checks, and viewport
fit-frame targeting.

`runtime::fit_view` now uses the shared `ViewportFitFrame` and `CanvasBounds`. `runtime::utils`
node bounds and inside-rect helpers now use the same `CanvasBounds`, and the old private
`runtime::utils::geometry` duplicate was deleted.

- `cargo nextest run -p jellyflow-runtime runtime::fit_view`: passed, 4 tests.
- `cargo nextest run -p jellyflow-runtime runtime::tests::utils`: passed, 10 tests.
- `cargo check -p jellyflow-runtime`: passed.

## Review

Self-review against the `review-workstream` checklist found no blocking or important findings.
The module is intentionally crate-private until the geometry contract stabilizes across later
JGS tasks.
