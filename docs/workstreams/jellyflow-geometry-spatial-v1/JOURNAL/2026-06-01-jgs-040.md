# JGS-040 Edge Endpoint Geometry

Status: DONE
Date: 2026-06-01

## Summary

Published `runtime::geometry` for renderer-neutral endpoint geometry while keeping internal
`CanvasBounds` helpers crate-private.

Added:

- `HandlePosition`
- `HandleBounds`
- `EdgeEndpointInput`
- `EdgeEndpointPosition`
- `EdgePosition`
- `handle_anchor_position`
- `handle_center_position`
- `edge_position`

The contract mirrors XyFlow's handle-position math without DOM measurement. Adapters provide
node-local handle rects and get canvas-space endpoint points plus side metadata.

Added `NodeGraphLookups::connection_for_edge` so adapters can get source/target node and port ids
for an edge without reading raw maps.
