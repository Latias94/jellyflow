# JVE-020 - Visible Node Runtime Contract

Date: 2026-06-02

Implemented the renderer-neutral visible node id contract in `runtime::rendering`.

Result:

- Added `VisibleNodeIdsRequest`.
- Added `resolve_visible_node_ids`.
- Added `NodeGraphStore::visible_node_ids(viewport_size)`.
- Covered viewport culling, disabled culling, hidden nodes, invalid viewport size, node-origin
  fallback, fallback size, and public surface reachability.

Validation:

- `cargo fmt --check`
- `cargo nextest run -p jellyflow-runtime visible_node`
- `cargo nextest run -p jellyflow-runtime --test public_surface`
