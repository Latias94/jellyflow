# JCF-020 - Fixture Vocabulary

Status: DONE
Date: 2026-06-01

## Summary

Added `jellyflow_runtime::runtime::conformance` as the first public headless fixture vocabulary.
The schema covers graph/view/config setup, trace recording configuration, runtime actions, gesture
actions, and normalized expected trace events.

## Implementation Notes

- Added `ConformanceScenario`, `ConformanceSetup`, `ConformanceTraceConfig`,
  `ConformanceAction`, `ConformanceTraceEvent`, `ConformanceViewChange`, and
  `ConformanceCallbackEvent`.
- Added schema version constant `CONFORMANCE_FIXTURE_SCHEMA_VERSION`.
- Made runtime gesture events and connection callback payloads serde-friendly so fixture traces can
  round-trip without renderer dependencies.
- Added public surface coverage that builds and serializes a node drag fixture.
