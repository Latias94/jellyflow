# JCF-030 - Fixture Runner

Status: DONE
Date: 2026-06-01

## Summary

Added the first headless conformance runner for `ConformanceScenario`. The runner executes fixture
actions against a real `NodeGraphStore`, records normalized store/gesture/callback traces, and
returns compact per-index trace mismatches.

## Implementation Notes

- Added `run_conformance_scenario` and `ConformanceRunner`.
- Added `ConformanceRunReport`, `ConformanceTraceMismatch`, and `ConformanceRunError`.
- Runner supports dispatching transactions, applying node drag, setting viewport/selection, and
  emitting gesture events.
- Trace recording uses the same normalized event shape as the fixture vocabulary.
- Callback recording mirrors the existing private interaction harness ordering.
